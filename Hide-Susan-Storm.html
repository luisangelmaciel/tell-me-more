Para el diseño responsivo suelo ocupar tres medidas para pantallas:

  • pequeñas de maximo 600px,
  • medianas de minimo 601px a maximo 992px y
  • grandes de minimo 993px

    • Para hacerlo un poco divertivo y no olvidar esta valiosa instrucción, decidi bautizarla con el nombre de Susan Storm (Uy, wow, que original. Lease en tono de voz de Sheldon Cooper de la Teoria de BigBang)

    Debe ser horrible ser temido, ser atacado, solo porque eres diferente, ¡solo porque eres un extraño en un mundo extraño! Susan Storm

    .susan-storm

    /* small */
    @media (max-width:600px) {
          .susan-storm-small {
            display: none!important
         }
    }

    /* medium */
    @media (max-width:992px) and (min-width:601px) {
          .susan-storm-medium {
            display: none!important
         }
    }

    /* large */
    @media (min-width:993px) {
          .susan-storm-large {
            display: none!important
         }
    }
